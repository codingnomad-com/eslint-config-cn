# eslint-config-cn

## Usage

### Installation

```bash
yarn add @codingnomad/eslint-config-cn
```

### .eslintrc.js

```js
module.exports = {
  extends: '@codingnomad/eslint-config-cn',
  rules: {
    ...
  },
};

```

### commitlint.config.js

```js
module.exports = {
  extends: ['@codingnomad/eslint-config-cn'],
};
```

### prettier.config.js

```js
module.exports = require('@codingnomad/eslint-config-cn/prettier.config');
```

### stylelint.config.js

```js
module.exports = {
  extends: ['@codingnomad/eslint-config-cn'],
};
```
